Installation
------------

This library is installable via [Composer](https://getcomposer.org/).
Just define the following requirement in your `composer.json` file:

    {
        "require": {
            "brick/translation": "0.1.*"
        }
    }

Requirements
------------

This library requires PHP 7.1 or later.
